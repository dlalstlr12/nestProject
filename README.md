## 프로젝트 소개
NestJS를 이용한 프로젝트입니다.
NestJS를 이용하여 OAuth 회원가입을 구현한 프로젝트, 웹소켓을 이용한 실시간 채팅방 구현 프로젝트, NestJS 환경 변수 설정 프로젝트를 구현했습니다.
Auth와 Chat 프로젝트들은 간단한 html로 테스트를 구현하였습니다.
